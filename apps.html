FlameAI – AI Assistant for Campus Resources

Project Description:

FlameAI is an AI-powered information assistant that uses GPT-4o and LangChain to provide students, staff, and visitors with real-time, context-aware answers about Claremont Graduate University. The system combines Retrieval-Augmented Generation (RAG), prompt engineering, and real-time web search to deliver structured support and recommendations.

Key Features:

  • Conversational AI agent with GPT-4o, using LangChain and LangGraph workflows
  • Live information retrieval using Tavily Search API
  • Session-level feedback integration with LangSmith for traceability and scoring
  • Streamlit frontend for fast deployment and user engagement

ACE Chatbot – AI-Powered Career Coach

Project Description:

ACE is an intelligent agentic chatbot designed to support university students through personalized resume reviews, interview prep, and job-market insights. The system uses Retrieval-Augmented Generation (RAG) and LangChain agents to deliver real-time, customized support.

Key Features:

  • LLM agent framework using LangChain + OpenAI GPT-4o
  • Dynamic prompt engineering for role-specific support
  • Integrated PGVector for semantic search (in progress)
  • Streamlit-based UX with LangSmith feedback tracing

Resume Evaluator – Virtual Tech Career Coach

Project Description:

A web-based AI assistant that offers actionable feedback on resumes and career alignment using job descriptions and personal data. Built to guide tech students and early-career professionals through job prep with smart LLM prompts and real-time feedback loops.

Key Features:

  • Personalized resume analysis using GPT-4o
  • Job-role alignment via prompt templates and LangChain tools
  • Live web search integration for market-relevant recommendations
  • LangSmith-driven feedback and evaluation tracking

Sakila Data Warehouse – Movie Rental Analytics

Project Description:

A complete ETL pipeline and data warehouse solution built on the Sakila database. Designed to support business intelligence reporting for rental performance, customer trends, and store metrics.

Key Features:

  • Star schema modeling with fact and dimension tables (e.g., Fact_rental, Dim_customer)
  • ETL automation with Apache Airflow + SQL Server
  • Data transformations for ratings, staff history, and unified location data

Tissue Suitability Analytics – Data Engineering Pipeline

Project Description:

An enterprise-ready SQL-based pipeline for transforming and analyzing donor tissue records. Built to support predictive modeling and regulatory screening within healthcare systems.

Key Features:

  • Normalized data ingestion from multiple source systems
  • Feature engineering for tissue-screening suitability
  • Validation logic to ensure consistency, integrity, and record traceability
